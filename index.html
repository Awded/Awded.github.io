Awded
Awded is a visual overlay for the deaf and hard of hearing players to visualize direction of audio during gameplay, for increased situational awareness.
Open Source

Awded is an open source project. The community can help improve and get involved in developing features.

If you find a bug or think of useful features, or even want your theme added, please post an issue on Github!

Works with Most Games

Awded is designed to read audio directly from Windows, so it is application agnostic. The only limiting factor is that applications must be windowed, or windowed-fullscreen

You can even simply use it as a music visualizer if you're not looking for accessibiilty features.

Customizeable

Awded is drivin by web technologies and leverages CSS variables, so every detail can be easily and maticulously changed.

Create a cool theme? Add it to the default available themes with a pull request or an issue on Github!
